Protocol Layer: WebSocket + Binary RPC + Protobuf

  • miku_sha1.h/c - SHA-1 hash (FIPS 180-4) for WebSocket handshake

  • miku_pb.h/c - Protobuf wire format codec: varint, svarint (zigzag), fixed32/64, bytes, string, bool; reader with skip support
  • 10 new tests: SHA1 (abc, empty), WS frame encode+decode, WS masked frame, WS handshake, RPC header codec, RPC message roundtrip, PB varint+string+bool, PB svarint, PB fixed32/64
  • 37 tests total, all passing
